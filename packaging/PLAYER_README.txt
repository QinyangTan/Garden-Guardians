Garden Guardians — portable copy
=================================

Run the game from THIS folder so it can find the "assets" folder next to the program.

  • Windows: double-click GardenGuardians.exe
  • macOS / Linux: in a terminal, run  ./GardenGuardians

macOS security: if the system says the app could not be verified, Control-click
GardenGuardians in Finder, choose Open, then Open again. Or System Settings →
Privacy & Security → Open Anyway after a blocked launch.

Your progress is saved in the "save" folder created here.
